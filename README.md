# LaserTrade
LaserTrade - Machine Learning market price prediction.

The project was created for ML training on financial data. The goal is not perfect prediction, but understanding the pipeline.

Technologies used
Python, Pandas, Scikit-learn, Matplotlib, yfinance, Streamlit

How to run
pip install -r requirements.txt
streamlit run app.py

Application
fill in the ticker (stock symbol) in the appropriate field and press enter

Results
after 5-10 seconds, a chart with historical market data and a prediction for tomorrow, whether it will go up or down, will appear
on the screen you will see not only the price, but also the indicators that traders follow

The model captures historical data, but its accuracy has limits
